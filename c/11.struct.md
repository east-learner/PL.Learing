##### [1.结构体](#top) <b id="target1"></b>
`在C语言中，可以使用结构体（Struct）来存放一组不同类型的数据。结构体的定义形式为：`

```c
struct 结构体名{
    结构体所包含的变量或数组
};
```

`例子`
```c
struct stu{
    char *name;  //姓名
    int num;  //学号
    int age;  //年龄
    char group;  //所在学习小组
    float score;  //成绩
};
```
